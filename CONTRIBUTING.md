# Contributing

To contribute to this project, you must submit a pull request. Pull requests will be completed in new branches and will not be merged until they have been reviewed by the team. 

### Branch Protection
Pull requests are required before merging into main. Only one approval is required before merging, and main is the only protected branch. Go wild!

### Code of Conduct
Generally, please try to be patient with me, as I am the only one contributing to this project at the moment. Baseline, don't be rude.

### How can I contribute?
Please feel free to report bugs, submit feature requests, and tackle open issues. 

When contributing, please adhere to the style guidelines used by ruff, mypy, and pylint, and please use these tools to resolve any style issues before submitting a pull request.

There is a test suite of 10 tests built into the project. Please make sure you run this and the tests pass before submitting your pull request.
